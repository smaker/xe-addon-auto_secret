개요
---
이 소프트웨어는 글 수정/작성 시 자동으로 비밀글으로 설정할 수 있는 애드온입니다.

이 소프트웨어는 GPL 을 채택한 오픈 소스 소프트웨어(OSS)로써 자유롭게 사용할 수 있는 자유 소프트웨어입니다. 이 라이선스의 조건을 준수하는 조건하에 누구나 자유롭게 사용할 수 있으며 개작 할 수 있습니다.
